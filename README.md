node-FRC
=======
### The [NodeJS][] ⇆ [FRC][] bridge


`node-FRC` exposes a variety of FRC functionality to node.js scripts.

Installation
------------

Install using `npm`, of course!

``` bash
$ npm install FRC
```

Or add it to the `"dependencies"` section of your `package.json` file.
